## Conway's Game of Life

See the [Wikipedia page](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life) about this.

### Run

```bash
python visualize.py
python visualize.py -n 20 -d 0.1 -f animation  # saves GIF to figures/
```
